Controller Programming Software (Learn More about Programming)

All software was scanned by VirusTotal before being listed. After downloading, you should exercise caution and scan it again with the tool of your choice. Send links or contact information for other software that works for programming the controller.

USB to Serial Driver

  • This driver is not necessary for the USB adapter. Microsoft's driver works fine under Windows 10. You will know it works when it appears as a COM port in Device Manager after plugging it into your computer.

EM-2 Series Software

  • If you have issues running this software, the publisher suggests you change your system locale to China. This doesn't appear to be the case for most users.
  • EM_V3-40
  • EM_V3-50
  • Factory 2021 Metacycle Settings - Just in case you needed a backup but didn't make one. It was created with EM V3-40. Note: the last 7 lines of this file could be any random numbers; see the serial protocol's Config Settings 7 for info.
