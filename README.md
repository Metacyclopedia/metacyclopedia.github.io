# Metacyclopedia

All about the Metacycle from Sondors.

## Getting started

This project uses [bun](https://bun.sh/) as the package manager. If you don't have it installed, you can get it from the [official website](https://bun.sh/).

You also will need [git lfs](https://git-lfs.github.com/) to clone the repository, as it contains large files. You can install it from the [official website](https://git-lfs.github.com/).

```
# Clone the repository
git clone https://github.com/Metacyclopedia/metacyclopedia.github.io.git

# Change into the project directory
cd metacyclopedia

# Install dependencies
bun install

# Start the development server
bun dev

# Open your browser and navigate to http://localhost:4321
```
